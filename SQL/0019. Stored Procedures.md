## ðŸ§ª Example: Increase Salary by Percentage

> ðŸŽ¯ Goal: Create a stored procedure that **increases the salary** of employees in a specific department by a given percentage.

### âœ… SQL Code:
```sql

   @Department NVARCHAR(50),
   @IncreasePercentage FLOAT
AS
BEGIN
   UPDATE Employees
   SET salary = salary + (salary * @IncreasePercentage / 100)
   WHERE department = @Department;

   WHERE department = @Department;
END;
```
