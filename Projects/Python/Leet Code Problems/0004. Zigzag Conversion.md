## ü™° Zigzag Conversion

> Given a string `s`, write the code that will convert it to a zigzag pattern with a given number of rows (`numRows`) and then read it line by line.

---

### üí° Example 1:
```
Input: s = "PAYPALISHIRING", numRows = 3
```
Visual Pattern:
```
P   A   H   N  
A P L S I I G  
Y   I   R      
```

‚úÖ Output: `"PAHNAPLSIIGYIR"`

---

### üí° Example 2:
```
Input: s = "PAYPALISHIRING", numRows = 4
```
Visual Pattern:
```
P     I    N
A   L S  I G
Y A   H R
P     I
```

‚úÖ Output: `"PINALSIGYAHRPI"`

---

### üí° Example 3:
```
Input: s = "A", numRows = 1
```

‚úÖ Output: `"A"`

---

### üìå Constraints

- `1 <= s.length <= 1000`
- `1 <= numRows <= 1000`
- `s` consists of English letters, `,`, and `.`

---
---

## üß† Python Solution: Zigzag Simulation

```python
class Solution:
    def convert(self, s: str, numRows: int) -> str:
        # Special case: no zigzag possible
        if numRows == 1 or numRows >= len(s):
            return s

        # Initialize a list of strings for each row
        rows = [''] * numRows
        current_row = 0
        going_down = False

        for c in s:
            rows[current_row] += c
            # Flip direction at the top or bottom row
            if current_row == 0 or current_row == numRows - 1:
                going_down = not going_down
            current_row += 1 if going_down else -1

        # Combine all rows
        return ''.join(rows)
```
