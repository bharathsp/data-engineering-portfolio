| Use Case                | Benefit                                                                 |
| ----------------------- | ----------------------------------------------------------------------- |

### âš ï¸ **Important Notes**

* You can specify user in **four ways**:

* The user must exist **inside the container**. If not, the command may fail with "user not found".
