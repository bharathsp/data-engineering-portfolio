### üîπ Python API Usage

Using APIs in Python typically involves sending HTTP requests to web services and processing the responses. This is a **core skill** in data engineering, web development, and automation.

---

## ‚úÖ 1. **Basic API Request using `requests`**

Install the `requests` library (if not already installed):

```bash
pip install requests
```

### üî∏ Example: GET Request

```python
import requests

url = 'https://api.example.com/data'
response = requests.get(url)

print(response.status_code)      # HTTP status code
print(response.json())           # JSON content (if response is JSON)
```

---

## ‚úÖ 2. **POST Request with Payload**

Used when you want to send data (e.g., for login or form submission).

```python
url = 'https://api.example.com/login'
data = {'username': 'user', 'password': 'pass'}

response = requests.post(url, json=data)  # or use `data=data` for form-encoded

print(response.json())
```

---

## ‚úÖ 3. **Headers and Authentication**

```python
headers = {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
}

response = requests.get('https://api.example.com/secure-data', headers=headers)
print(response.json())
```

---

## ‚úÖ 4. **Handling API Response**

```python
if response.status_code == 200:
    data = response.json()
    # Process data
else:
    print("Error:", response.status_code, response.text)
```

---

## ‚úÖ 5. **Saving API Response to File (CSV/JSON)**

### Save JSON to file

```python
import json

with open('data.json', 'w') as f:
    json.dump(response.json(), f, indent=4)
```

### Save to CSV (from JSON)

```python
import pandas as pd

data = response.json()
df = pd.DataFrame(data)
df.to_csv('data.csv', index=False)
```

---

## üß† Best Practices

* ‚úÖ Handle errors (`try-except`, check `status_code`)
* ‚úÖ Use authentication for protected APIs
* ‚úÖ Use pagination if the API supports it (`page=1`, `limit=100`, etc.)
* ‚úÖ Respect API rate limits (check headers like `X-RateLimit-Remaining`)
* ‚úÖ Cache or store results to avoid re-hitting APIs

## ‚öôÔ∏è Real-world Example: OpenWeather API

```python

api_key = 'your_api_key'
city = 'London'
url = f'https://api.openweathermap.org/data/2.5/weather?q={city}&appid={api_key}'
```
