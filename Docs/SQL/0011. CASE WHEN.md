### ðŸ” Explanation

- `CASE` âž¤ Begins the conditional logic block

- `ELSE result_default` âž¤ Value returned if none of the conditions match (optional)  
- `END` âž¤ Closes the `CASE` block

### ðŸŽ“ Example

Suppose we have a table called `students` with the columns:  
- `student_id`  
- `name`  
- `score`  

We want to **categorize students based on their score** into:
- Excellent
- Good
- Average
- Needs Improvement

```sql
SELECT 
    student_id,
    name,
    score,
    CASE 
        WHEN score >= 90 THEN 'Excellent'
        WHEN score >= 75 THEN 'Good'
        WHEN score >= 50 THEN 'Average'
        ELSE 'Needs Improvement'
    END AS performance_category
FROM 
    students;
```
