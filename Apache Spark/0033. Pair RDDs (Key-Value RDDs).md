# ğŸ”‘ **Pair RDDs (Key-Value RDDs)**

## ğŸ§© What is a Pair RDD?

* A **Pair RDD** is just an RDD where each element is a **key-value pair (K, V)**.
* Keys are used for grouping, reducing, or joining data.
* Values are the associated data items.

ğŸ‘‰ Formally:

```
RDD[(K, V)]
```

where `K = key`, `V = value`.

---

## âš¡ Why Pair RDDs?

* They enable **distributed key-based operations**, like in **MapReduce**.
* Spark provides special transformations/actions optimized for Pair RDDs.

Examples:

* `reduceByKey()` â†’ Combine values with the same key.
* `groupByKey()` â†’ Group values by key.
* `join()` â†’ Join two RDDs by key.
* `sortByKey()` â†’ Sort by key.

---

## ğŸ“Š Example: Creating a Pair RDD

```python
from pyspark import SparkContext
sc = SparkContext("local", "PairRDDExample")

data = [("apple", 2), ("banana", 3), ("apple", 4)]
rdd = sc.parallelize(data)

print(rdd.collect())
```

**Output:**

```
[('apple', 2), ('banana', 3), ('apple', 4)]
```

Here, `rdd` is a **Pair RDD** with key = fruit, value = number.

---

## ğŸ“Š Example: `reduceByKey()`

Sum values for each key:

```python
result = rdd.reduceByKey(lambda a, b: a + b)
print(result.collect())
```

**Output:**

```
[('banana', 3), ('apple', 6)]
```

---

## ğŸ“Š Example: `groupByKey()`

Group values by key:

```python

print(grouped.collect())
```

**Output:**

```
[('banana', [3]), ('apple', [2, 4])]
```

---

## ğŸ“Š Example: `sortByKey()`

```python

print(sorted_rdd.collect())
```

**Output:**

```
[('apple', 2), ('apple', 4), ('banana', 3)]
```

# ğŸ”‘ **keys() and values() in Spark**

## ğŸ§© What are they?

* **`keys()`** â†’ Returns an RDD containing only the keys from a Pair RDD.
* **`values()`** â†’ Returns an RDD containing only the values from a Pair RDD.

ğŸ‘‰ They are **transformations** (lazy operations) and produce a new RDD.

---

## âš¡ Syntax

```python
rdd.keys()
rdd.values()
```

---

## ğŸ“Š Example: Using `keys()` and `values()`

```python
from pyspark import SparkContext
sc = SparkContext("local", "KeysValuesExample")

data = [("apple", 2), ("banana", 3), ("apple", 4)]
rdd = sc.parallelize(data)

keys_rdd = rdd.keys()
values_rdd = rdd.values()

print("Keys:", keys_rdd.collect())
print("Values:", values_rdd.collect())
```

**Output:**

```
Keys: ['apple', 'banana', 'apple']
Values: [2, 3, 4]
```

---

## ğŸ“Š Example: Counting Keys

You can use `keys()` with transformations like `distinct()`:

```python

print(unique_keys.collect())
```

**Output:**

```
['apple', 'banana']
```

---

## ğŸ“Š Example: Summing Values

Instead of manually extracting values, you can just call:

```python

print(total)
```

**Output:**

```
9
```

# ğŸ”„ Summary

* `keys()` â†’ Extracts **all keys** from a Pair RDD.
* `values()` â†’ Extracts **all values** from a Pair RDD.
* Both return new RDDs, often used before `distinct()`, `countByValue()`, or aggregations.
