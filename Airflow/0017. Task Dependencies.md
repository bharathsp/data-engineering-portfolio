### ğŸ” **Multiple Dependencies:**

#### Parallel Tasks:

```python
task1 >> [task2, task3]
```

This means:

* `task1` runs first
* Then `task2` and `task3` run **in parallel**

#### Join Tasks:

```python
[task2, task3] >> task4
```

This means:

* `task4` runs **only after** both `task2` and `task3` are completed
