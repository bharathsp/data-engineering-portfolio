## âœ… Python Solution

```python
def roman_to_int(s: str) -> int:
    roman_map = {
        'I': 1,    'V': 5,    'X': 10,
        'L': 50,   'C': 100,  'D': 500,
        'M': 1000
    }
    
    total = 0
    prev = 0
    
    for char in reversed(s):
        value = roman_map[char]
        if value < prev:
            total -= value
        else:
            total += value
            prev = value
    return total
```
